# my-first-Java-program

### Five Facts about Java:

1) Developed in 1995 by James Gosling at Sun Microsystems;<br>

2) It was initially called `Oak`in hommage to an oak tree in front of Gosling's office. Eventually the name changed to `Green` and finally, `Java`, that came from Java coffee, hence the logo.<br>
<br>
<img src="https://cdn.worldvectorlogo.com/logos/java.svg" width="100" />
<br>

3) There are four editions of Java for building different applications:<br>
a) **Standard Edition (SE)**<br>
b) **Enterprise Edition (EE)** for building very large scale and distributed systems.<br>
It's build on top of Java SE and provides additional libraries.<br>
c) **Micro Edition (ME)** designed for mobile devices<br>
d) **Java Card** used in smart cards.
<br>

4) Nine million of Java developers worldwide. 

5) Java was originally created by Sun Microsystems' "Green Team" to program embedded systems and electronic appliances, like set-top boxes and interactive televisions, focusing on the "Write Once, Run Anywhere" (WORA) principle for diverse hardware.
